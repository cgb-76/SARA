The wiki/log.md format (from sara-update Step 3) is a table row:
| [[MTG-001]] | 2026-04-27 | meeting | transcript.md | [[REQ-001]], [[DEC-002]], [[ACT-003]] |

To find entity IDs for ingest item {N}:
1. Read wiki/log.md
2. Find the row(s) where the first column contains [[{N}]] (e.g. [[MTG-001]])
3. Parse the last column for entity ID wikilinks: extract IDs from [[REQ-001]], [[DEC-002]], etc.
4. Use those IDs to read wiki pages (same as before)

Why log.md not plan.md: plan.md contains placeholder IDs (REQ-NNN) at write time;
by the time sara-minutes runs (stage=complete), the actual IDs are in wiki/log.md.

Pitfall 7: Do NOT use plan.md placeholder IDs as real entity IDs.
